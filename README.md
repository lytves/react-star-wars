# react-star-wars ⚫🤖⚔️

Project is based on the youtube video course [React Course / Redux Practice](https://www.youtube.com/watch?v=euYBnQ5MbLM) by Dev Pandaren

### Init Project: ⭐🔫👸
```
npx create-react-app react-star-wars
cd react-star-wars
npm start
```
